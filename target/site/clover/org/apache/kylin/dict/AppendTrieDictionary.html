46    /**
47    * A dictionary based on Trie data structure that maps enumerations of byte[] to
48    * int IDs, used for global dictionary.
49    * <p>
50    * Trie data is split into sub trees, called {@link AppendDictSlice}.
51    * <p>
52    * With Trie the memory footprint of the mapping is kinda minimized at the cost
53    * CPU, if compared to HashMap of ID Arrays. Performance test shows Trie is
54    * roughly 10 times slower, so there's a cache layer overlays on top of Trie and
55    * gracefully fall back to Trie using a weak reference.
56    * <p>
57    * The implementation is NOT thread-safe for now.
58    * <p>
59    * TODO making it thread-safe
60    *
61    * @author sunyerui
62    */
